# Finance

Personal finance tooling.

- **`app/`** — Flutter Android app (expenses, budgets, credit-card cycles, overspending alerts, insights) backed by Supabase. See [app/README.md](app/README.md) for setup.
- **`supabase/`** — SQL migrations for the Supabase backend (schema, RLS, alert triggers).
- **`index.html`** — BankDash, a standalone web dashboard for analysing bank statements (PDF/CSV/XLSX).
- **`meridian.html`** — Meridian, a local-first companion web app.
